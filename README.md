# CSS Master - Application d'Apprentissage Interactif

Une application React moderne pour apprendre le CSS de manière interactive avec des exercices pratiques, un éditeur avec surbrillance syntaxique, et un suivi de progression.

## 🚀 Fonctionnalités

### Éditeur Avancé
- **Monaco Editor** avec surbrillance syntaxique pour HTML et CSS
- Thème sombre professionnel
- Numérotation des lignes
- Retour automatique à la ligne

### Cours Structurés
- **6 modules** progressifs couvrant tous les aspects du CSS
- **23 leçons** avec exercices pratiques
- Validation en temps réel des solutions
- Aperçu live des résultats

### Feuille de Route d'Apprentissage

#### 📚 Module 1: Bases du CSS
1. **CSS en ligne (Inline)** - Styles directement dans les balises HTML
2. **CSS interne (Internal)** - Styles dans la balise `<style>`
3. **CSS externe (External)** - Fichiers CSS séparés
4. **Sélecteurs de type** - Ciblage des balises HTML
5. **Sélecteurs de classe** - Utilisation des classes CSS
6. **Sélecteurs d'identifiant** - Utilisation des IDs

#### 🎯 Module 2: Sélecteurs Avancés
7. **Sélecteurs descendants** - Ciblage des éléments imbriqués
8. **Pseudo-classes :hover** - Effets au survol
9. **Sélecteurs d'attributs** - Ciblage par attributs HTML
10. **Combinaisons de sélecteurs** - Sélecteurs multiples

#### 📐 Module 3: Le Modèle de Boîte
11. **Marges (margin)** - Espacement extérieur
12. **Padding et bordures** - Espacement intérieur et contours
13. **Dimensions et box-sizing** - Contrôle des tailles

#### 🎮 Module 4: Positionnement
14. **Position relative** - Déplacement relatif
15. **Position absolue** - Positionnement absolu
16. **Position fixe** - Éléments fixes

#### ⭐ Module 5: Flexbox
17. **Introduction à Flexbox** - Concepts de base
18. **justify-content et align-items** - Alignement des éléments
19. **flex-direction et flex-wrap** - Direction et retour à la ligne
20. **Flex-grow, flex-shrink, flex-basis** - Contrôle de la flexibilité

#### 🏆 Module 6: CSS Grid
21. **Introduction à CSS Grid** - Grilles bidimensionnelles
22. **grid-template-rows et grid-gap** - Lignes et espacement
23. **Positionnement dans la grille** - Placement des éléments

### Suivi de Progression
- Barre de progression globale
- Indicateurs visuels pour les leçons complétées
- Compteur de leçons terminées
- Validation automatique des exercices

### Interface Utilisateur
- Design moderne avec Tailwind CSS
- Thème sombre professionnel
- Icônes Lucide React
- Animations fluides
- Responsive design

## 🛠️ Technologies Utilisées

- **React 18** - Framework JavaScript
- **TypeScript** - Typage statique
- **Vite** - Outil de build rapide
- **Tailwind CSS** - Framework CSS utilitaire
- **Monaco Editor** - Éditeur de code avancé
- **Lucide React** - Bibliothèque d'icônes

## 🚀 Installation et Utilisation

1. **Cloner le projet**
   ```bash
   git clone <repository-url>
   cd cours-css
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer l'application**
   ```bash
   npm run dev
   ```

4. **Ouvrir dans le navigateur**
   ```
   http://localhost:5173
   ```

## 📖 Comment Utiliser

1. **Navigation** : Utilisez la barre latérale pour naviguer entre les modules et leçons
2. **Édition** : Modifiez le code HTML et CSS dans les onglets dédiés
3. **Validation** : Les exercices se valident automatiquement
4. **Aperçu** : Visualisez le résultat en temps réel dans le panneau de droite
5. **Progression** : Suivez votre avancement avec la barre de progression

## 🎯 Objectifs d'Apprentissage

À la fin de ce cours, vous saurez :
- Maîtriser les différentes méthodes d'intégration du CSS
- Utiliser tous types de sélecteurs CSS
- Comprendre et appliquer le modèle de boîte
- Positionner les éléments avec précision
- Créer des layouts flexibles avec Flexbox
- Construire des grilles complexes avec CSS Grid
- Écrire du CSS maintenable et organisé
